| Problem | Key Twist |
|---------|-----------|

| **Find Pivot Index** (724) | Left sum == Right sum |
| **Product of Array Except Self** (238) | Prefix product from both ends |
| **Range Sum Query 2D - Immutable** (304) | 2D prefix sum |
| **Maximum Size Subarray Sum Equals k** (325) | Store first occurrence for longest length |
| **Contiguous Array** (525) | Convert 0â†’-1, find longest sum=0 |
| **Path Sum III** (437) | Prefix sum on tree paths |

## Problem Selection by Technique

### Prefix Sum + Hashmap (Count)
- Subarray Sum Equals K
- Subarray Sums Divisible by K
- Binary Subarrays With Sum

### Prefix Sum + Hashmap (First Index for Length)
- Maximum Size Subarray Sum Equals k
- Contiguous Array
- Make Sum Divisible by P

### Prefix Sum + Monotonic Deque
- Shortest Subarray with Sum at Least K

### Prefix Sum + Merge Sort / BIT
- Count of Range Sum
- Reverse Pairs (similar technique)

### 2D Prefix Sum
- Range Sum Query 2D
- Number of Submatrices That Sum to Target
- Max Sum of Rectangle No Larger Than K

### Prefix Sum + DP
- Maximum Sum of 3 Non-Overlapping Subarrays
- Split Array With Same Average
